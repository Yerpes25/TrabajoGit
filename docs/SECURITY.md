# Seguridad

## Roles
- admin
- technician
- client

## Permisos (resumen)
- Admin: gestión completa + validación
- Técnico: gestiona sus partes, no valida
- Cliente: solo lectura de sus partes finalizadas/validadas

## Auditoría
Registrar:
- login/logout
- cambios de saldo
- cambios de estado de partes
- ediciones/borrados
- importaciones

Retención mínima: 5 años.
